# robust-rl-locomotion

Study of PPO robustness under deployment-time observation and dynamics shift in continuous control.

Focus:
- Evaluation-time perturbations only
- Deterministic evaluation protocol
- Multi-seed reproducibility
- Fixed metrics schema
- No training-time robustness modifications (v1.0)

Environment:
- HalfCheetah-v4

Release target:
v1.0-robust-shift
